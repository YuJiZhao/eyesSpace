# 三：关于运维

项目运行在实体服务器上，从咸鱼淘来的n手Dell T110服务器
+ 1c8g，1T500G
+ 操作系统：Centos7
+ 前端：Nginx
+ 后端：Tomcat
+ 内网穿透：Sakura Frp
